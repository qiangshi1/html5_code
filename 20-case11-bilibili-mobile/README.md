## easyless vscode的一个插件 less转css
1. settings中加这句话设置导出路径
```css
    "less.compile": {
        "out": "../css/"
    },
```

## px2vw,  vscode的一个插件 px转vw
1. 配置：
    - px2vw.width 设计宽度, 保持默认即可
    - px2vw.toFixedNum 保留小数, 本例是375
